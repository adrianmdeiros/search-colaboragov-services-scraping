## Base URL: https://services-links.vercel.app/

### Endpoints
`POST` /api/search  

#### Example
`Body` 
```json
{
  "message": "Férias"
}
```
`Response`
```json
{
  "services": [
    {
        "name": "Consultar Férias"
        "link": "https://catalogo-colaboragov.sistema.gov.br/servico/consultar-ferias"
    }
  ]
}
```
